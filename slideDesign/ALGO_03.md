## スライド 2：導入 ― 「ハノイの塔」で遊ぼう

- 一番端の柱に円盤を積み上げる（上にいくほど小さい順）
- 円盤を一回につき一個動かして、他の柱に移動させる
- **ルール**
  1. 円盤は一個ずつしか動かせない
  2. 大きい円盤を小さい円盤の上に置けない
- 初めと反対側の端の柱にそっくり移し替えられたら完成！

## スライド 9：数式の展開 ②

```
sum(n) = 1 + 2 + … + (n-1) + n
       = { 1 + 2 + … + (n-1) } + n
```

---

## スライド 10：数式の展開 ③ — sum(n-1) の発見

```
sum(n) = 1 + 2 + … + (n-1) + n
       = { 1 + 2 + … + (n-1) } + n
              ↑
           sum(n - 1)
```

---

## スライド 11：数式の展開 ④ — 再帰式

```
sum(n) = 1 + 2 + … + (n-1) + n
       = { 1 + 2 + … + (n-1) } + n
              ↑
           sum(n - 1)

       = sum(n - 1) + n
```

---

## スライド 12：再帰の定義

> **再帰** → あるものの定義に、それ自身が含まれていること

```
sum(n) = sum(n - 1) + n
```

※数学で言う「漸化式」と言ったほうが馴染みある？

## スライド 16：sum(n) — 再帰版（完成）

```c
int sum(int n)
{
    if (n == 1) { return 1; }  // ← 追記
    return sum(n-1) + n;
}
```

## スライド 17：再帰の注意点

**終了条件（ベースケース）を記載しないと、無限に続く**

例：`if (n == 1) { return 1; }`

---

## スライド 18：再帰の良い点

- 定義がそのままコードに表すことができる
- コードの記述量が少なくなる

---

## スライド 19：演習課題

C言語にて、**n の階乗を計算する関数 `factorial(int n)`** を定義し、
階乗の計算結果を示すプログラムを作成せよ。

**要件**

- プログラムが実行されたら、整数 n の入力を促すこと
  （何も知らない実行者が次のアクションを起こせる必要がある）
- `factorial` 関数は再帰関数とする
- 最後に階乗の計算結果を示すこと

---

## スライド 20：factorial(n) — 解答例

```c
int factorial(int n)
{
    if (n == 1) { return 1; }
    return factorial(n-1) * n;
}
```
