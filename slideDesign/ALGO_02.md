## スライド 20 — 課題：配列への挿入関数

**C言語の文字配列にて、指定位置への挿入関数 `insert(char *str, char c, int n)` を定義せよ。**

【要件】
- プログラムが実行されたら、初期の文字列を表示させること
- ユーザーに挿入位置と挿入する文字入力を促すこと（何も知らない実行者が次のアクションを起こせる必要がある）
- 最後に挿入後の文字列を示すこと

---

## スライド 21 — 解答例：指定位置への挿入関数

```c
void insert(char *str, char c, int n)
{
    int len = strlen(str);

    // 右にずらす（末尾から）
    for (int i = len; i >= n; i--) {
        str[i + 1] = str[i];
    }
    // 挿入
    str[n] = c;
}
```

## スライド 34 — スタックとキュー（説明）

| | スタック | キュー |
|--|----------|--------|
