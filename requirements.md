# 要件定義書：アルゴリズム講義スライドシステム

## 1. プロジェクト概要

アルゴリズムの講義（全15回）で使用するWebベースのスライドプレゼンテーションシステム。Reactで構築し、テクニカルなデザインをライトテーマで実現する。

## 2. 技術スタック

| 項目 | 技術 |
|------|------|
| フレームワーク | React |
| ルーティング | React Router |
| スタイリング | CSS Modules or Styled Components |
| コードハイライト | Prism.js または highlight.js |
| 数式レンダリング | KaTeX |
| PDFエクスポート | react-to-print または html2pdf.js |
| アニメーション | Framer Motion または CSS Transitions |

## 3. ページ構成・ルーティング

| パス | 内容 |
|------|------|
| `/` | トップページ（授業回一覧） |
| `/:lessonId` | 特定の授業回のスライド表示（例：`/1` で第1回） |
| `/styles` | スタイルセット一覧ページ |

## 4. スライド表示機能

### 4.1 全画面表示

- スライドはビューポート全体を使って全画面表示する
- アスペクト比を固定し、リサイズ時もレイアウトを維持する

### 4.2 ナビゲーション

#### マウス操作

- 画面右側クリック → 次のスライドへ進む
- 画面左側クリック → 前のスライドへ戻る

#### キーボード操作

- `→` / `Space` → 次のスライドへ進む
- `←` → 前のスライドへ戻る
- `Esc` → サムネイル一覧の表示/非表示トグル

### 4.3 サムネイルナビゲーション

- 画面下部にマウスカーソルを移動すると、スライド一覧のサムネイルバーがスライドアップで表示される
- 各サムネイルをクリックすると該当スライドへジャンプする
- 現在のスライドがハイライト表示される

### 4.4 アニメーション

- スライド内要素のフェードイン等、段階的な表示に対応する
- クリック／キー操作でアニメーションステップを進める
- すべてのアニメーションステップ完了後、次のクリックで次スライドへ遷移する

## 5. スライドコンテンツ

### 5.1 コンテンツ管理方式

- 各授業回のスライドはReactコンポーネントとして直接記述する
- ディレクトリ構成例：
  ```
  src/
    lessons/
      lesson01/
        Slide01.jsx
        Slide02.jsx
        ...
        index.js   // スライド一覧のエクスポート
      lesson02/
        ...
  ```

### 5.2 対応コンテンツタイプ

| タイプ | 説明 |
|--------|------|
| テキスト・見出し | H1〜H4、本文、リスト |
| コードブロック | シンタックスハイライト付き（主要言語対応） |
| 数式 | LaTeX記法によるインライン・ブロック数式 |
| 画像・図表 | 画像埋め込み、図表の配置 |
| アニメーション | フェードイン等の段階的表示 |

### 5.3 想定規模

- 授業回数：15回
- 1回あたりのスライド枚数：約50枚

## 6. スタイルセット

### 6.1 デザイン方針

- **テーマ**：ライトテーマ × テクニカルスタイル
- 参考：[re-presentation.jp](https://re-presentation.jp/tool/3D_presentation_5.html) のテクニカル・構造的なレイアウトを踏襲しつつ、配色をライトテーマに反転
- モノスペースフォントの活用、グリッド・ワイヤーフレーム的な装飾要素
- アルゴリズム講義にふさわしい、構造的で知的な印象

### 6.2 統一スタイルコンポーネント

全15回・全スライドで以下のスタイルを統一する：

- **カラーセット**：背景色、テキスト色、アクセントカラー、コードブロック背景色など
- **タイポグラフィ**：見出し（H1〜H4）、本文、コード、数式のフォント・サイズ・ウェイト
- **基本コンポーネント**：
  - ボックス（情報ボックス、注意ボックス、定義ボックスなど）
  - コードブロック
  - 数式ブロック
  - 画像コンテナ
  - リスト（番号付き・番号なし）
  - テーブル
  - 区切り線・装飾要素

### 6.3 スタイル一覧ページ（`/styles`）

- 上記すべてのスタイルコンポーネントをサンプル付きで一覧表示する
- カラーパレット、フォントサンプル、各コンポーネントの実装例を確認できる
- スタイルガイド／デザインシステムとして機能する

## 7. PDFエクスポート機能

- 特定の授業回を指定してPDFとしてエクスポートできる
- スライドのレイアウト・スタイルを可能な限り維持する
- エクスポートボタンをUI上に配置する（サムネイルバー付近等）

## 8. 非機能要件

- スライド切り替えは60fps以上でスムーズに動作すること
- 主要ブラウザ（Chrome, Firefox, Safari, Edge）の最新版で動作すること
- レスポンシブ対応（プロジェクター投影・ノートPC表示を想定）
